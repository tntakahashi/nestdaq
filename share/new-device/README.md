# nestdaq new-device command

## Usage
```
Usage: new-device [options]

Options:
  -p,--prefix        Perfix that points to the directory path to which files are output
                     (default = <current working directory>/my-nestdaq-device)
  -d,--device        Device name. 
                     If there are multiple devices, use "A;B;C" like in a cmake list.
  -h,--help          Show this help message
```
